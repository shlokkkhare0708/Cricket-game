🏏 Bat Ball Stump Game

A simple browser-based Bat Ball Stump Game built using HTML, CSS, and JavaScript. The game allows the player to compete against the computer, with the computer making a random choice each round. The application tracks wins, losses, and ties, and stores the score using the browser's Local Storage, ensuring the score persists even after refreshing the page.

✨ Features
Interactive game with three choices:
🏏 Bat
⚾ Ball
🪵 Stump
Random computer move generation.
Automatic result calculation (Win, Lose, or Tie).
Live score tracking.
Persistent score using localStorage.
Reset button to clear saved scores.
Image-based buttons for a better user experience.

🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
Browser Local Storage API

🎮 How It Works
The player selects Bat, Ball, or Stump.
The computer randomly selects one of the three options.
The game compares both choices and determines the winner.
The score is updated and displayed on the screen.
The score is saved in the browser's local storage, so it remains available even after reloading the page.
Clicking the Reset button clears the stored score and starts a new game.
